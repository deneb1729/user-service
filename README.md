# User Service with Node, Express, Sequelize and Postgres

Description: This repo is an example project for user management.

Main Stack:

-   Node
-   Express
-   Sequelize
-   Postgres

Requirements:

-   node
-   npm

For to check node and npm versions, run:

```sh
# node version
node -v

# npm version
npm -v
```

# Annexes

## Annex A: develop environment

```sh
npm install
npm run dev
```
